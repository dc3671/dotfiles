# Dotfiles #

My dotfiles for:

* vim(with spf13-vim)
* tmux
* zsh(with oh-my-zsh)

## Install ##

```
git clone https://github.com/dc3671/dotfiles.git
cd dotfiles

rm ~/.vimrc.local
ln -s ~/.vimrc.local $PWD/.vimrc.local

rm ~/.tmux.conf
ln -s ~/.tmux.conf $PWD/.tmux.conf

rm ~/.zshrc
ln -s ~/.zshrc $PWD/.zshrc
```
